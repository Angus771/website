MocaccinoOS is a minimal Linux meta-Distribution

{{< blocks/link-down color="info" >}}
{{< /blocks/cover >}} {{% blocks/lead color="primary" %}} MocaccinoOS uses [luet](https://luet-lab.github.io/docs) as Package Manager which is completely static, and it comes with two variants: * Mocaccino Micro: Musl-based Linux From Scratch using luet. Suited for cloud and docker, but more to come as a Desktop flavor. * Mocaccino Portage: Gentoo-based (former Sabayon branch) suited for Desktop environments {{% /blocks/lead %}} {{< blocks/section color="dark" >}} {{% blocks/feature icon="fa-lightbulb" title="No frills" %}} MocaccinoOS uses a layered approach for common packages, smoother upgrades so you can focus on the things that matters. {{% /blocks/feature %}} {{% blocks/feature icon="fab fa-github" title="Contributions welcome!" url="https://github.com/MocaccinoOS" %}} MocaccinoOS is a Work in progress, [step over by our GitHub Organization](https://github.com/MocaccinoOS).
